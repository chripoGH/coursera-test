Access index.html, e.g.:
https://chripogh.github.io/coursera-html-css-javascript/module2-solution/

Start broswer sync server via cmd, e.g. for module2-solution:
> cd (...)\repo\coursera-html-css-javascript\module2-solution
> browser-sync start -s -f "*" "css/styles.css"

or better:
> browser-sync start -s -f "**/*"
